# 感情壞蛋記錄簿

一個匿名分享感情故事的 Next.js 應用程式。

## 功能特色

- 📝 匿名故事投稿
- 📖 故事瀏覽與閱讀
- 👤 像素藝術頭像生成
- 🔒 管理後台審核
- 📱 響應式設計
- 🎨 時尚雜誌風格

## 開始使用

1. 安裝依賴：
```bash
npm install
```

2. 設定環境變數：
```bash
cp .env.example .env.local
# 編輯 .env.local 設定 ADMIN_PASSWORD
```

3. 啟動開發伺服器：
```bash
npm run dev
```

4. 訪問 [http://localhost:3000](http://localhost:3000)
